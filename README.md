# theexperts (replica-stub)

هذه نسخة تطويرية مبدئية تحاكي تجربة موقع كتالوج/ضيافة مع:
- Smooth scrolling (Lenis)
- Scroll & entrance animations (GSAP + ScrollTrigger)
- Hover + tilt effects (GSAP)
- Page transition scaffold (Barba.js)

كيف تجرب محليًا:
1. ضع الملفات في مجلد واحد (index.html, styles.css, main.js).
2. افتح `index.html` في المتصفح أو استعمل خادم محلي بسيط (Live Server).

ملاحظات قانونية:
- استخدمت روابط صور من الموقع الهدف كـ placeholders مؤقتة للاختبار — ستحتاج لاستبدالها بصور لديك حقوق استخدامها قبل النشر النهائي.
